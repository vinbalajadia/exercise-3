## Project Overview
This is a simple **To-Do List application in Python** that allows you to:
- **Add** tasks  
- **Show** all tasks  
- **Remove** tasks  

The app runs in the terminal and is designed for beginners learning Python basics such as lists, loops, and conditionals.
